# Inclusive STEAM AI 🌍🤝

Barchaga moslashtirilgan, inkluziv STEAM ta'lim platformasi. Ushbu loyiha yoshlar va turli ehtiyojlarga ega bo'lgan o'quvchilar uchun maxsus ishlab chiqilgan.

## 🚀 Asosiy Imkoniyatlar

1.  **🧠 Aqlli Onboarding:** Kirishda foydalanuvchining ehtiyojlarini tahlil qiladi va eng qulay interfeysni tavsiya etadi.
2.  **✋ Imo-ishora (Gesture) Interfeysi:** Sun'iy intellekt yordamida qo'l harakatlari orqali darslarni boshqarish.
3.  **🎧 Audio Interfeys (Ko'zi ojizlar uchun):** To'liq ovozli navigatsiya va tabiiy o'zbek tilidagi audio yordamchi.
4.  **📱 Mobil Moslashuv:** Platforma barcha turdagi smartfon va planshetlarda to'liq ishlaydi.

## 🛠 Texnologiyalar

*   **React + Vite** (Frontend framework)
*   **MediaPipe** (Imo-ishoralarni aniqlash uchun AI)
*   **Web Speech API** (Ovozni aniqlash va gapirish uchun)
*   **CSS3 (Modern UI/UX)** (Glassmorphism va premium dizayn)

## 📦 O'rnatish va Ishga tushirish

1.  Repository-ni yuklab oling:
    ```bash
    git clone https://github.com/USERNAME/inclusive-ai-steam.git
    ```
2.  Kerakli kutubxonalarni o'rnating:
    ```bash
    npm install
    ```
3.  Dasturni lokal ishga tushiring:
    ```bash
    npm run dev
    ```
4.  Dasturni build (tayyor holga keltirish) qilish:
    ```bash
    npm run build
    ```
